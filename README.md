# RS-Clone

<h3>How To Use:</h3>

<ol>
    <li>clone this repo</li>
    <li>cd rsclone/</li>
    <li>npm i</li>
    <li>cd client/</li>
    <li>rm - rf .git</li>
    <li>npm i</li>
</ol>
<hr>

<h4>Scripts:</h4>

<ul>
<li>npm run server - run server</li>
<li>npm run client - run client</li>
<li>npm run dev - run both</li>
</ul>
